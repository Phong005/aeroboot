                           动态产生 VS 解决方案
                           
     When you have a *nix project, you may want to load it in VS2005 IDE, you can
 use this tool to generate a .vcproj file to server for this.
 
     很多时候，我们需要在 VS2005 中查看 *nix 的工程，比如 grub2。如果我们手工添加一个
 个文件，过滤器，那是一场噩梦。这个工具就是用来解决这个问题的。
 
     e.g. c:\grub_code>unix2vs.exe "c:\grub_code\grub2"
