
    Rom_App: Create a build-in romfs for Avlgo
    ================================================================

        This package is used for create a functional build-in
    romfs for Avlgo. Avlgo will use it to support ISOEmu/Memdisk.

        To use it, put your gziped files to `Input` dir, and run 
    make-romfs_ntfs.bat. Finally copy the `romfs.bin` to ../stage2.
